Installation
===================

``pyskeb`` is currently hosted on PyPI. It required python >= 3.7.

You can simply install pyskeb from PyPI with the following command:

.. code:: shell

    pip install pyskeb

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/narugo1992/scrit.git@main

of ``pyskeb`` should be shown.
